## Dai Yanan · 戴亚楠 —— 声音 / 音频大模型

同济大学 · 从音频表示到可控声音生成，做一整套「声音大模型」的地基与上层。

我把声音大模型拆成**自下而上的四层**，四个开源项目正好对应这四层：

> **① 表示** —— [resona](https://github.com/LX588387/resona) · 2022–2023
> 通用音频自监督表示学习：掩码音频建模 / 对比预训练，语音 · 音乐 · 环境声统一表示。
>
> **② 编解码** —— [soniq](https://github.com/LX588387/soniq) · 2023–2024
> 神经音频编解码与离散化：低比特率编解码器 + 残差矢量量化，连续音频 → 离散 token。
>
> **③ 理解** —— [auris](https://github.com/LX588387/auris) · 2024–2025
> 音频语言模型：把音频编码器 / 离散 token 接入 LLM，做音频理解、描述、问答与检索。
>
> **④ 生成** —— [vocca](https://github.com/LX588387/vocca) · 2025–2026
> 可控声音生成大模型：文本 / 提示驱动的语音与通用音频生成，情感 / 风格 / 音色可控。

**工程约定**：纯 NumPy 参考实现、`import` 不拉深度学习框架、无 GPU / 无网络也能跑核心与单测；需要训练时再接可选 PyTorch 后端。离线优先，CI 全绿。
